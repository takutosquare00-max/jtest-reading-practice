J.TEST F級・G級 リーディング速度向上練習教材 第2回（1時間）

授業の目標

  • より高度な文法項目（動詞のない形～助詞）を素早く読む練習をする
  • 中程度の長さの文章を読む練習を始める
  • 複数の文法項目を含む文章を理解する力を身につける

復習：前回のポイント確認（5分）

前回学習した文法項目（1-9）を素早く復習しましょう。

  • 名詞述語文、イ形容詞、ダ形容詞、動詞のます形、副詞形、存在文、疑問詞、動詞のます形を使った表現、動詞のて形

第1部：短文リーディング練習（文法項目10-18）（35分）

10. 動詞のない形を使った表現

練習10-1：禁止・否定

  1. すみませんが、ここでたばこを吸わないでください。
  2. シャワーを浴びないで、寝ました。
  3. 朝ご飯を食べないで、学校に行きました。
練習ポイント：「ない形」の作り方と「～ないでください」（禁止）を素早く認識する

11. 動詞のた形を使った表現

練習11-1：列挙・後

  1. 日曜日は本を読んだり、テレビを見たりします。
  2. 顔を洗った後で、出かけます。
  3. ご飯を食べた後で、勉強します。
練習ポイント：「～たり～たりします」（列挙）と「～た後で」（後）を素早く認識する

12. 動詞の辞書形を使った表現

練習12-1：前・こと

  1. 出かける前に、顔を洗います。
  2. 仕事は日本語を教えることです。
  3. 寝る前に、歯を磨きます。
練習ポイント：「～前に」（前）と「～こと」（名詞化）を素早く認識する

13. 普通体

練習13-1：動詞の普通体

  1. 私は毎日日本語を勉強する。
  2. 昨日は本を読んだ。
  3. 明日は学校に行かない。
  4. 先週は映画を見なかった。

練習13-2：イ形容詞の普通体

  1. このりんごはおいしい。
  2. 昨日は暑かった。
  3. この本は高くない。
  4. 先週は忙しくなかった。

練習13-3：名詞・ダ形容詞の普通体

  1. 私は会社員だ。
  2. これは本だ。
  3. 田中さんは有名だ。
  4. 昨日は日曜日だった。
  5. これは本ではない。
  6. 昨日は忙しくなかった。
練習ポイント：普通体と丁寧体の違いを素早く認識し、普通体の文を理解する

14. 普通体を使った表現

練習14-1：思う・言う

  1. この花はとてもきれいだと思います。
  2. リーさんはまた日本に来ると言いました。
  3. 明日は晴れるでしょう。
  4. リーさんは日本語が上手でしょう。

練習14-2：時・から

  1. 京都へ行った時にこれを買いました。
  2. 子供の時は野菜が嫌いでした。
  3. 暇な時は海へ行きます。
  4. 今朝は早く学校に行きました。テストがあったからです。
練習ポイント：「～と思います」「～と言います」「～時」「～から」を素早く認識する

15. 自動詞と他動詞

練習15-1：自動詞・他動詞

  1. ドアが開きます。
  2. ドアを開けます。
  3. 電気がつきます。
  4. 電気をつけます。
  5. 窓が閉まります。
  6. 窓を閉めます。
練習ポイント：「が」+自動詞と「を」+他動詞の違いを素早く見分ける

16. 比較の表現

練習16-1：より・方が

  1. これはあれより新しいです。
  2. ここはあそこより静かです。
  3. あれよりこれの方が新しいです。
  4. あそこよりここの方が静かです。
  5. これとあれとどちらの方が新しいですか。
  6. こことあそことどちらの方が静かですか。
練習ポイント：比較の表現（より、方が、どちらの方が）を素早く認識する

17. 助詞

練習17-1：は

  1. 私は会社員です。
  2. 私は9時から5時まで働きます。
  3. 日曜日は働きません。
  4. 土曜日は働きますが、日曜日は働きません。
  5. 教室には誰もいません。
  6. 日本語は夜勉強します。

練習17-2：が

  1. 犬がいます。
  2. 山田さんが来ました。
  3. 犬が好きです。
  4. 日本語が上手です。
  5. 車がほしいです。
  6. お茶が飲みたいです。
  7. 田中さんは目がきれいです。
  8. これはいいカメラですが、高いです。
  9. 部屋が涼しくなりました。

練習17-3：を

  1. ご飯を食べます。
  2. 教室を出ます。
  3. 電車を降ります。
  4. 道を歩きます。
  5. 公園を散歩します。
  6. 会社の前を通ります。
  7. 鳥が空を飛んでいます。
  8. 会社を休みます。
  9. 水をください。
  10. 部屋を涼しくしました。

練習17-4：の

  1. 私の本です。
  2. 日本語の本です。
  3. この本は私のです。
  4. 私のはこれです。
  5. 白いのは1000円です。
  6. 日本語が上手なのはリーさんです。
  7. 一番背の高い人は田中さんです。

練習17-5：で

  1. 学校で勉強します。
  2. ペンで書きます。
  3. 電車で来ました。
  4. バスで行きます。
  5. 木で机を作ります。
  6. 牛乳でアイスクリームを作ります。
  7. 病気で会社を休みました。
  8. りんごは3つで500円です。
  9. りんごは全部で1000円です。

練習17-6：と

  1. りんごとバナナがあります。
  2. 友達と一緒に昼ご飯を食べました。
  3. 友達と会います。
  4. 先生と話しました。
  5. この花はとてもきれいだと思います。
  6. リーさんはまた日本に来ると言いました。
  7. これは天麩羅と言う料理です。

練習17-7：や・など

  1. りんごやバナナがあります。
  2. りんごやバナナなどがあります。
  3. 本や雑誌などがあります。

練習17-8：も

  1. 田中さんも会社員です。
  2. 私も田中さんも会社員です。
  3. 教室には誰もいません。
  4. 山田さんは教室にもトイレにもいません。

練習17-9：に

  1. 教室に学生がいます。
  2. 教室に机があります。
  3. 明日も学校に来ますか。
  4. 明日も学校に行きますか。
  5. 家に帰ります。
  6. 電車に乗ります。
  7. 教室に入ります。
  8. 友達に会います。
  9. 毎朝6時に起きます。
  10. 一週間に一回掃除します。
  11. デパートへ服を買いに行きます。
  12. 田中さんは医者になりました。
  13. 子供を医者にしたいです。
  14. コーヒーに砂糖を入れます。
  15. 葉書に切手を貼ります。
  16. 本に名前を書きます。
  17. 私はあのアパートに住んでいます。

練習17-10：へ

  1. 明日も学校へ来ますか。
  2. 明日も学校へ行きますか。
  3. 家へ帰ります。

練習17-11：から・まで

  1. 京都から来ました。
  2. 9時から働きます。
  3. 今朝はテストがありましたから、早く学校に行きました。
  4. 今朝は早く学校に行きました。テストがあったからです。
  5. 京都から東京まで3時間かかります。
  6. 9時から5時まで働きます。

練習17-12：か

  1. お茶かコーヒーをください。
  2. 田中さんは来るか来ないか分かりません。
  3. 山田さんは元気か元気ではないか分かりません。
  4. 何か飲み物をください。
  5. それは何ですか。
  6. 明日も学校へ来ますか。
  7. 山田さんは会社員ですか、学生ですか。
  8. 一緒に映画を見ませんか。
練習ポイント：各助詞の様々な用法を文脈から素早く判断する

18. 終助詞・副詞・その他

練習18-1：終助詞

  1. 暑いですね。
  2. 大丈夫ですよ。
  3. 私も行くわ。

練習18-2：位・頃

  1. 家から学校まで20分位かかります。
  2. 山田さんは2時頃来ました。

練習18-3：だけ・しか

  1. 田中さんだけ来ました。
  2. ひとつだけあります。
  3. 田中さんしか来ませんでした。
  4. ひとつしかありません。

練習18-4：とても・大変

  1. このりんごはとてもおいしいです。
  2. 今朝はとても早く来ました。
  3. この部屋はとてもきれいです。
  4. このりんごは大変おいしいです。
  5. それは大変ですね。

練習18-5：あまり・よく

  1. このりんごはあまりおいしくないです。
  2. この部屋はあまりきれいではありません。
  3. 肉はあまり食べません。
  4. 暇な時はよく海へ行きます。
  5. 日本語がよく分かりません。

練習18-6：その他の表現

  1. 田中さん達は帰りましたよ。
  2. 今日は一日中忙しかったです。
  3. 田中さんはもう帰りましたよ。
  4. 田中さんはもう来ています。
  5. もう飲み物がありません。
  6. 田中さんはまだ教室にいます。
  7. 田中さんはまだ来ていません。
練習ポイント：副詞や終助詞の意味を素早く理解し、文全体の意味を把握する

第2部：中程度の長さの文章（20分）

練習2-1：日常生活

文章1
私は毎朝6時に起きます。起きた後で、顔を洗って、歯を磨きます。それから、朝ご飯を食べます。朝ご飯はパンとコーヒーです。朝ご飯を食べてから、会社へ行きます。会社は家から電車で30分かかります。会社では9時から5時まで働きます。仕事は日本語を教えることです。仕事が終わった後で、家に帰ります。家に帰ってから、晩ご飯を食べます。晩ご飯を食べた後で、テレビを見たり、本を読んだりします。11時頃に寝ます。
問題
  1. 私は何時に起きますか。
  2. 朝ご飯を食べる前に何をしますか。
  3. 会社は家からどのくらいかかりますか。
  4. 仕事は何ですか。
  5. 家に帰ってから何をしますか。
文章2
田中さんは会社員です。田中さんは有名でおもしろい人です。田中さんの目はきれいです。田中さんは毎日日本語を勉強しています。日本語が上手になりたいからです。田中さんは暇な時はよく海へ行きます。海は静かで、とてもきれいです。田中さんは海で泳いだり、散歩したりします。海は家から車で1時間かかります。田中さんは車を持っています。車は新しくて、とてもきれいです。
問題
  1. 田中さんはどんな人ですか。
  2. 田中さんはなぜ日本語を勉強していますか。
  3. 田中さんは暇な時はどこへ行きますか。
  4. 海は家からどのくらいかかりますか。
  5. 田中さんの車はどんな車ですか。
文章3
昨日は日曜日でした。私は友達と一緒にデパートへ買い物に行きました。デパートにはたくさんの店があります。私は服を買いたかったです。でも、服は高くて、あまり買いませんでした。友達は本を買いました。買い物をした後で、レストランで昼ご飯を食べました。レストランはきれいで、静かでした。昼ご飯を食べてから、映画を見に行きました。映画はおもしろかったです。映画を見た後で、家に帰りました。
問題
  1. 昨日は何曜日でしたか。
  2. 私はどこへ行きましたか。
  3. 私は何を買いましたか。
  4. レストランはどんなレストランでしたか。
  5. 映画を見た後で何をしましたか。

まとめ（5分）

今日学習した文法項目（10-18）と中程度の文章の読み方を復習しましょう。

次回は、より長い文章と実践的な問題形式を練習します。

宿題

次回までに、第2回で学習した中程度の文章（文章1-3）を、それぞれ3回ずつ素早く読む練習をしてください。時間を測って、読む速度が上がっているか確認しましょう。

解答例

文章1
  1. 6時に起きます。
  2. 顔を洗って、歯を磨きます。
  3. 電車で30分かかります。
  4. 日本語を教えることです。
  5. テレビを見たり、本を読んだりします。
文章2
  1. 有名でおもしろい人です。
  2. 日本語が上手になりたいからです。
  3. 海へ行きます。
  4. 車で1時間かかります。
  5. 新しくて、とてもきれいです。
文章3
  1. 日曜日でした。
  2. デパートへ行きました。
  3. あまり買いませんでした。
  4. きれいで、静かでした。
  5. 家に帰りました。
