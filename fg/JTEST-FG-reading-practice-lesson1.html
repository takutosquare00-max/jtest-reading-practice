J.TEST F級・G級 リーディング速度向上練習教材 第1回（1時間）

授業の目標

  • リーディング速度向上の基本テクニックを理解する
  • 基本的な文法項目（名詞述語文～動詞のて形）を素早く読む練習をする
  • 短文を正確に、素早く理解する力を身につける

導入：リーディング速度向上のコツ（10分）

リーディング速度向上のコツ

  • 音読しない：頭の中で音読せず、視覚的に理解する
  • 戻り読みをしない：一度読んだ部分に戻らない
  • 文の構造を素早く把握する：主語・述語・修飾語を瞬時に見分ける
  • 語彙を文脈で推測する：知らない語彙があっても文全体から意味を推測する
  • 時間を測る：常に時間を意識して読む

練習の進め方

  1. 各短文を素早く読む（音読しない）
  2. 文の構造（主語・述語）を素早く見つける
  3. 時間を意識して読む練習をする

第1部：短文リーディング練習（文法項目1-9）（40分）

1. 名詞述語文

練習1-1：現在形

  1. 私は会社員です。
  2. 田中さんは学生です。
  3. これは本です。
  4. あれは机です。
  5. それは何ですか。
  6. どれがあなたの傘ですか。
  7. 私は会社員ではありません。
  8. 田中さんは会社員ではありませんか。

練習1-2：過去形

  1. 私は去年まで学生でした。
  2. 田中さんは会社員ではありませんでした。
  3. 昨日は日曜日でしたか。
  4. おとといは火曜日でした。

練習1-3：並列文

  1. 私は会社員で、田中さんは学生です。
  2. これは本で、それは雑誌です。
  3. 今日は月曜日で、明日は火曜日です。
練習ポイント：主語（名詞+は）と述語（名詞+です）を素早く見つける

2. イ形容詞を使った文

練習2-1：現在形

  1. このりんごはおいしいです。
  2. この部屋は明るいです。
  3. 今日は暑いです。
  4. この本は新しいです。
  5. このりんごはおいしくないです。
  6. 今日は寒くないです。

練習2-2：過去形

  1. 昨日の天気はよかったです。
  2. 昨日はいい天気でした。
  3. 昨日の天気はよくなかったです。
  4. 先週は忙しかったです。

練習2-3：並列文

  1. このりんごは安くておいしいです。
  2. この部屋は広くて明るいです。
  3. この本は安くてきれいです。

練習2-4：名詞化

  1. おいしいのはこれです。
  2. 白いのは安いです。
  3. 赤いのはきれいです。
練習ポイント：イ形容詞の語尾（い）に注意し、否定形（くない）や過去形（かった）を素早く認識する

3. ダ形容詞を使った文

練習3-1：現在形

  1. この花はきれいです。
  2. 田中さんは有名です。
  3. この部屋は静かです。
  4. この花はきれいではありません。
  5. これはきれいな花です。
  6. 田中さんは有名な人です。

練習3-2：過去形

  1. 田中さんは有名でした。
  2. 昨日は静かでした。
  3. 先週は忙しくなかったです。

練習3-3：並列文

  1. 田中さんは有名でおもしろいです。
  2. この部屋は静かできれいです。

練習3-4：名詞化

  1. 有名なのは田中さんです。
  2. きれいなのは高いです。
  3. 静かなのは嫌です。
練習ポイント：ダ形容詞の「な」の有無に注意し、名詞を修飾する時は「な」が入ることを素早く認識する

4. 動詞のます形

練習4-1：基本形

  1. ご飯を食べます。
  2. 本を読みます。
  3. 日本語を勉強します。
  4. 会社で働きます。
  5. 昨日は本を読みませんでした。
  6. 明日も学校に行きます。

練習4-2：目的語と場所

  1. 教室で日本語を勉強します。
  2. 図書館で本を読みます。
  3. レストランでご飯を食べます。
  4. 公園で散歩します。
練習ポイント：「を」と「で」の違いを素早く見分ける（「を」=目的語、「で」=場所）

5. 副詞形を使った表現

練習5-1：イ形容詞の副詞形

  1. 早く行きます。
  2. 部屋が涼しくなります。
  3. 部屋を涼しくします。
  4. 静かに話します。

練習5-2：ダ形容詞の副詞形

  1. 上手に話します。
  2. 部屋がきれいになります。
  3. 部屋をきれいにします。
  4. 静かに座ります。
練習ポイント：副詞形（イ形容詞は「く」、ダ形容詞は「に」）を素早く認識する

6. 存在を表す文

練習6-1：人や動物

  1. 田中さんは教室にいます。
  2. 教室に学生がいます。
  3. 教室に誰がいますか。
  4. 庭に何がいますか。
  5. 教室には誰もいません。

練習6-2：物

  1. 机は教室にあります。
  2. 教室に机があります。
  3. 教室に何がありますか。
  4. 本棚に本があります。
練習ポイント：「います」（人・動物）と「あります」（物）の違いを素早く見分ける

7. 疑問詞を使った表現

練習7-1：疑問詞+が/を

  1. 教室に誰がいますか。
  2. 何を飲みますか。
  3. どこに行きますか。
  4. いつ来ますか。

練習7-2：疑問詞+か/も

  1. 誰かいますか。
  2. 何か飲みますか。
  3. 誰もいません。
  4. 何も飲みません。
練習ポイント：疑問詞の後に続く助詞（が/を/か/も）によって意味が変わることを素早く理解する

8. 動詞のます形を使った表現

練習8-1：誘い・希望

  1. 一緒に映画を見ませんか。
  2. 一緒に映画を見ましょう。
  3. 映画が見たいです。
  4. 映画は見たくないです。

練習8-2：同時動作・目的

  1. お茶を飲みながら話しましょう。
  2. デパートへ服を買いに行きます。
  3. 図書館へ本を借りに行きます。
練習ポイント：「～ませんか」（誘い）、「～たいです」（希望）、「～ながら」（同時動作）、「～に行きます」（目的）を素早く認識する

9. 動詞のて形を使った表現

練習9-1：依頼・状態

  1. 明日は早く来てください。
  2. 薬を飲んでください。
  3. テレビを見ています。
  4. 結婚しています。
  5. 赤い服を着ています。

練習9-2：順序・理由

  1. 顔を洗ってから、出かけます。
  2. 歯を磨いて、顔を洗って、それから出かけます。
  3. ご飯を食べて、勉強します。
  4. はさみを使って、切ります。
  5. かぜをひいて、会社を休みました。
練習ポイント：「て形」の様々な用法（依頼、状態、順序、理由）を文脈から素早く判断する

まとめ練習（10分）

練習問題：以下の文章を素早く読んで、質問に答えてください

文章
私は毎朝6時に起きます。起きた後で、顔を洗って、歯を磨きます。それから、朝ご飯を食べます。朝ご飯はパンとコーヒーです。朝ご飯を食べてから、会社へ行きます。会社は家から電車で30分かかります。会社では9時から5時まで働きます。仕事が終わった後で、家に帰ります。家に帰ってから、晩ご飯を食べます。晩ご飯を食べた後で、テレビを見たり、本を読んだりします。11時頃に寝ます。
問題
  1. 私は何時に起きますか。
  2. 朝ご飯を食べる前に何をしますか。
  3. 会社は家からどのくらいかかりますか。
  4. 会社では何時から何時まで働きますか。
  5. 家に帰ってから何をしますか。

宿題

次回までに、第1回で学習した文法項目（1-9）の短文を、それぞれ10回ずつ素早く読む練習をしてください。時間を測って、読む速度が上がっているか確認しましょう。

解答例

まとめ練習の解答
  1. 6時に起きます。
  2. 顔を洗って、歯を磨きます。
  3. 電車で30分かかります。
  4. 9時から5時まで働きます。
  5. テレビを見たり、本を読んだりします。
